# 分享与公开访问

## 怎么创建分享链接

创建分享前，先切到正确的工作空间。
个人文件在个人空间里分享，团队文件在对应团队空间里分享。

在文件或文件夹的操作菜单里创建分享链接时，可以设置:

- 可选密码
- 过期时间
- 最大下载次数

页面里会提供常用过期选项，例如 1 小时、1 天、7 天、30 天或不过期。

## 文件分享和文件夹分享有什么区别

### 文件分享

适合发送单个文件。访问者打开链接后，可以直接预览或下载。

### 文件夹分享

适合发送一整套资料。公开页面支持:

- 浏览共享文件夹
- 进入子文件夹
- 面包屑返回上一级
- 预览文件
- 下载文件
- 在列表和网格之间切换

## 一条链接什么时候会失效

分享链接会在下面任一情况发生后失效:

- 到达过期时间
- 下载次数达到上限
- 你主动删除了这条分享
- 管理员在后台删除了这条分享

## 一个文件能不能同时有多条分享链接

当前版本里，同一个文件或文件夹同一时间只能保留一个活跃分享。

如果你想重新生成一条新的链接，有两种做法:

- 先删除旧链接，再创建新链接
- 等旧链接过期后再创建新链接

如果你想改密码、过期时间或下载次数，不需要换链接，直接在 `我的分享` 里编辑原来的分享即可。

## 密码保护怎么用

如果分享设置了密码，访问者需要先输入密码才能进入公开页面。
验证成功后，当前浏览器通常会在大约 1 小时内记住这次验证。

## 你自己怎么管理已经发出去的链接

普通用户可以在左侧 `我的分享` 页面里管理当前工作空间的分享:

- 复制分享链接
- 打开公开页面
- 编辑密码、过期时间和下载次数限制
- 删除不再需要的分享
- 查看访问次数和下载次数

## 管理员能做什么

管理员可以在 `管理 -> 分享` 里统一查看和删除全站分享链接。

常见用法:

- 某个公开链接不该继续访问
- 某个分享已经不再需要
- 想统一检查当前有哪些链接还在对外可用

# 用户手册

这份手册按普通用户登录后的实际页面来写。  
如果你已经拿到一个账号，照着下面的顺序看，基本就能把日常操作摸熟。

## 登录和首次进入

登录页不需要你先决定“这是登录还是注册”。  
输入用户名或邮箱后，页面会自动判断流程：

- 系统里还没有任何用户：创建管理员
- 输入的是已有账号：登录
- 输入的是新账号，而且管理员允许公开注册：注册普通账号

需要注意：

- 第一个创建成功的账号会自动成为管理员
- 后续新注册的普通账号，需要先点激活邮件才能登录
- 如果管理员关闭了公开注册，联系管理员给你创建账号
- 如果管理员已经配好邮件，而且你的邮箱已验证，可以在登录页直接申请重置邮件

## 先认识工作空间

登录后，左侧最上方会看到工作空间列表：

- `我的云盘`：你的个人文件、个人分享、个人回收站和 WebDAV 账号都在这里
- 团队空间：只有你加入团队后才会出现；每个团队空间都有自己的文件、分享和回收站

搜索、分享和回收站都会跟着当前工作空间走。  
要管理哪一组内容，先切到对应工作空间。

## 文件页面最常用的位置

- 左侧工作空间列表：切换个人空间或团队空间
- 左侧目录树：快速进入目标文件夹
- 顶部搜索框：搜索当前工作空间里的文件和文件夹
- `回收站`：处理当前工作空间里的已删除内容
- `我的分享`：查看当前工作空间里已经发出去的链接
- `WebDAV`：只在个人空间里出现，用来创建桌面客户端账号
- `设置`：调整个人资料、界面、安全和团队相关设置

## 上传和整理文件

文件列表、右键菜单和顶部操作区可以完成大多数日常工作：

- 创建文件夹
- 创建空白文本文件
- 上传文件
- 上传文件夹
- 下载文件
- 重命名、复制、移动、删除文件和文件夹
- 在列表视图和网格视图之间切换
- 按名称、大小、创建时间、更新时间或类型排序

你也可以直接拖拽：

- 把文件或文件夹拖到左侧目录树里的目标文件夹
- 把文件或文件夹拖到左侧的回收站

## 搜索、多选和批量处理

在当前工作空间里，你可以按名称搜索文件和文件夹。  
需要一次处理多个项目时，可以直接多选后批量执行：

- 批量移动
- 批量复制
- 批量删除

在文件列表和回收站里都可以使用 `Ctrl + A` 或 `Cmd + A` 全选当前页。

## 预览、编辑和版本历史

很多常见文件都可以直接在网页里打开，例如：

- 图片
- 音频和视频
- PDF
- Markdown
- CSV / TSV
- JSON
- XML
- 纯文本和常见代码文件

文本类文件通常可以直接编辑。保存后会自动生成历史版本。

你可以在版本历史里：

- 查看旧版本
- 恢复某个旧版本
- 删除不需要的旧版本

恢复旧版本后，那个版本之后的新版本记录会被截断，所以恢复前先确认。

如果保存时出现冲突提示，通常说明这份文件在你打开后又被别人改过了。先刷新内容，再决定是否继续保存。

## 回收站

普通删除不会立刻彻底清除文件或文件夹，而是先进入回收站。

你可以在回收站中：

- 恢复项目
- 永久删除单个项目
- 清空整个回收站
- 批量恢复
- 批量永久删除

如果原来的父目录已经不存在，恢复时项目会回到根目录。

## 分享链接

分享会跟着当前工作空间走。  
要分享团队空间里的文件，先切到对应团队空间再创建分享。

AsterDrive 支持文件分享和文件夹分享。创建分享时可以设置：

- 密码
- 过期时间
- 最大下载次数

需要注意的规则：

- 同一个文件或文件夹同一时间只能保留一条活跃分享
- 如果要改密码、过期时间或下载次数，不需要重新建链接，去 `我的分享` 里直接编辑即可
- 如果链接设置了密码，访问者在当前浏览器验证成功后，通常约 1 小时内不需要重复输入密码

## 我的分享

左侧 `我的分享` 页面会列出当前工作空间里已经发出去的链接。

你可以在那里：

- 复制分享链接
- 打开公开页面
- 编辑密码、过期时间和下载次数限制
- 删除不再需要的分享
- 查看访问次数和下载次数

## WebDAV

如果你希望在 Finder、Windows 资源管理器、rclone 或其他桌面工具里直接使用 AsterDrive，可以创建 WebDAV 专用账号。

WebDAV 账号在个人空间里管理，不跟团队空间共用。

常见做法是：

- 每台设备一个独立账号
- 某台设备丢失时单独停用
- 把某个账号限制到根目录下某个文件夹

创建 WebDAV 账号时可以：

- 自定义用户名
- 自定义密码，或让系统自动生成
- 指定访问范围

密码只会在创建成功时显示一次，建议立刻保存。

默认 WebDAV 地址通常是：

```text
https://你的域名/webdav/
```

如果管理员修改了 WebDAV 前缀，请改用新的地址。

## 设置

设置页分成四个部分。

### 个人资料

你可以修改：

- 展示名称
- 头像

头像支持两种常见来源：

- 上传自己的头像图片
- 使用根据邮箱生成的 Gravatar 头像

用户名在这里仍然是只读显示。  
邮箱会在这里显示，但真正的邮箱改绑在 `安全` 里处理。

### 界面主题

你可以调整：

- 浅色 / 深色 / 跟随系统
- 主题颜色
- 显示语言
- 文件浏览器默认视图

### 安全

这里不只改密码，还负责邮箱状态和邮箱改绑。

你可以看到：

- 当前邮箱是否已经验证
- 是否有一个“待确认的新邮箱”

如果当前邮箱已经验证，你可以：

- 输入一个新邮箱
- 发送确认邮件到新邮箱
- 在需要时重新发送确认邮件

新邮箱只有在你点开确认链接之后才会真正生效。

你也可以在这里修改登录密码。修改密码时需要先输入当前密码。

当前版本里，密码修改成功后，当前浏览器会话会继续保持登录。

如果你是从登录页进入“找回密码”，重置链接同样会发到已验证邮箱。

### 团队

`设置 -> 团队` 会列出你加入的团队。

你可以在这里：

- 查看团队名称、简介、成员数和空间占用
- 打开某个团队工作空间
- 查看已归档团队

如果你在团队里是管理员或所有者，还可以继续管理成员和查看团队审计。  
如果你是团队所有者，还可以归档团队。

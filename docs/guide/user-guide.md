# 用户手册

## 登录页怎么用

登录页不需要你先手动切换“登录”还是“注册”。
输入用户名或邮箱后，页面会自动判断当前应该做什么：

- 系统里还没有任何用户时：创建管理员
- 用户已存在且你输入的是已有账号：登录
- 用户已存在且你输入的是新账号：注册

第一个创建成功的账号会自动成为管理员。

## 登录后会看到什么

登录后，左侧是目录树和常用入口，右侧是当前目录内容。最常用的位置有：

- `Root`：你的根目录
- 左侧目录树：快速进入任意文件夹
- 顶部搜索框：搜索你自己账号下的文件和文件夹，输入后按 `Enter`
- `回收站`：恢复或彻底删除已删除项目
- `我的分享`：查看你已经发出去的公开链接
- `WebDAV`：创建桌面客户端要用的专用账号
- `设置`：调整个人资料、主题、颜色、语言和默认视图

## 新建、上传和整理文件

文件列表、右键菜单和顶部操作区就能完成大多数日常工作：

- 创建文件夹
- 创建空白文本文件
- 上传文件
- 上传文件夹
- 下载文件
- 重命名、复制、移动、删除文件和文件夹
- 列表 / 网格视图切换
- 搜索自己的文件和文件夹

你也可以直接拖拽：

- 把文件或文件夹拖到左侧目录树中的目标目录
- 把文件或文件夹拖到侧边栏的回收站

## 多选和批量操作

选中多个项目后，可以直接批量处理：

- 批量移动
- 批量复制
- 批量删除
- 批量恢复回收站项目
- 批量永久删除回收站项目

在文件列表和回收站里都可以使用 `Ctrl + A` 或 `Cmd + A` 全选当前列表。

## 预览、编辑和版本历史

很多常见文件都可以直接在网页里打开：

- 图片预览
- 音频和视频播放
- PDF 预览
- Markdown、CSV / TSV、JSON、XML 的格式化查看
- 文本和代码文件查看

支持直接编辑的通常是文本类文件，例如：

- Markdown
- CSV / TSV
- JSON
- XML
- 配置文件、脚本、日志和常见代码文件

保存后会自动生成历史版本。你可以在版本历史里：

- 查看旧版本
- 恢复某个版本
- 删除不需要的旧版本

恢复旧版本后，该版本及其之后的历史版本会一起被截断。

编辑文本时，AsterDrive 会自动处理锁定和冲突检查。
如果你保存时看到冲突提示，通常说明这份文件已经被别人改过，先刷新内容再决定是否继续。

## 回收站

普通删除不会立刻彻底清除文件或文件夹，而是先进入回收站。

你可以在回收站中：

- 恢复项目
- 永久删除单个项目
- 清空整个回收站
- 切换列表 / 网格视图
- 多选后批量恢复或批量永久删除

如果原来的父目录已经不存在，恢复时项目会回到根目录。

## 分享链接

AsterDrive 支持文件分享和文件夹分享。创建分享时可以设置：

- 密码
- 过期时间
- 下载次数限制

文件分享页面可以直接预览或下载单个文件。

文件夹分享支持：

- 浏览共享文件夹
- 进入子文件夹
- 预览文件
- 下载文件

同一个文件或文件夹同一时间只能保留一个活跃分享链接。需要新链接时，先删除旧链接，或者等旧链接过期。

如果链接设置了密码，访问者在当前浏览器验证成功后，通常约 1 小时内不需要重复输入密码。

你创建过的链接可以在左侧 `我的分享` 页面统一查看、复制和删除。

## WebDAV

如果你希望在 Finder、Windows 资源管理器、rclone 或其他桌面工具里直接使用 AsterDrive，可以创建 WebDAV 专用账号。

WebDAV 账号和网页登录账号分开管理，常见用法是：

- 每台设备一个独立账号
- 某台设备丢失时单独停用
- 把某个账号限制到根目录下某个文件夹

创建 WebDAV 账号时可以：

- 自定义用户名
- 自定义密码，或让系统自动生成
- 指定访问范围

密码只会在创建成功时显示一次，建议当场保存到密码管理器。

默认 WebDAV 地址是：

```text
https://你的域名/webdav/
```

如果管理员修改了 WebDAV 前缀，请改用新的地址。

## 设置

在设置页里，你可以调整：

- 展示名称
- 头像来源
- 主题模式
- 主题颜色
- 显示语言
- 文件浏览器默认视图

头像支持两种常见来源：

- 上传一张自己的头像
- 使用根据邮箱生成的 Gravatar 头像

这些设置只影响你的账号界面显示和浏览习惯，不会改动其他用户的数据。

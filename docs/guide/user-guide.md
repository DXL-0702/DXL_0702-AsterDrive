# 用户手册

AsterDrive 登录后的主要工作区可以分成六部分来看：文件管理、预览与编辑、回收站、分享、WebDAV 和个人设置。

## 登录后会看到什么

登录后，左侧是目录树和常用入口，右侧是当前目录内容。最常用的位置有：

- `Root`：你的根目录
- 目录树：快速进入任意文件夹
- 顶部搜索框：搜索你自己账号下的文件和文件夹
- `回收站`：恢复或彻底删除已删除项目
- `WebDAV`：创建桌面客户端要用的专用账号
- `设置`：调整主题、颜色、语言和默认视图

## 文件和文件夹

日常文件管理都可以在网页里完成：

- 创建文件夹
- 上传文件
- 下载文件
- 重命名、复制、移动、删除文件和文件夹
- 列表 / 网格视图切换
- 搜索自己的文件和文件夹

你也可以直接拖拽：

- 把文件或文件夹拖到左侧目录树中的目标目录
- 把文件或文件夹拖到侧边栏的回收站

## 多选和批量操作

当你需要一次处理很多项目时，可以使用多选。

支持的批量操作包括：

- 批量移动
- 批量复制
- 批量删除
- 批量恢复回收站项目
- 批量永久删除回收站项目

在文件列表和回收站里都可以使用 `Ctrl + A` 或 `Cmd + A` 全选当前列表。

## 预览、编辑和版本历史

很多常见文件都可以直接在网页里打开。常见场景包括：

- 图片预览
- 音频和视频播放
- PDF 预览
- Markdown、CSV / TSV、JSON、XML 的格式化查看
- 文本和代码文件查看

其中一部分文本文件还可以直接编辑。浏览器内编辑更适合：

- `text/*`
- `application/json`
- `application/xml`

编辑保存后会自动生成一个历史版本。你可以在版本历史里：

- 查看旧版本
- 恢复某个版本
- 删除不需要的旧版本

这套编辑能力适合文本类文件，不是多人协作编辑器。

## 回收站

删除文件或文件夹后，项目不会马上彻底消失，而是先进入回收站。

你可以在回收站中：

- 恢复项目
- 永久删除单个项目
- 清空整个回收站
- 切换列表 / 网格视图
- 多选后批量恢复或批量永久删除

如果原来的父目录已经不存在，恢复时项目会回到根目录。

回收站页面只用于恢复和删除，不提供文件预览。

## 分享链接

AsterDrive 支持文件分享和文件夹分享。创建分享时可以设置：

- 密码
- 过期时间
- 下载次数限制

文件分享适合直接预览和下载单个文件。

文件夹分享支持：

- 浏览共享文件夹
- 进入子文件夹
- 预览文件
- 下载文件

同一个文件或文件夹同一时间只能保留一个活跃分享链接。需要新链接时，先删除旧链接，或者等旧链接过期。

如果链接设置了密码，访问者在当前浏览器验证成功后，通常约 1 小时内不需要重复输入密码。

## WebDAV

如果你希望在 Finder、Windows 资源管理器、rclone 或其他桌面工具里直接使用 AsterDrive，可以创建 WebDAV 专用账号。

WebDAV 账号和网页登录账号分开管理，适合这些场景：

- 每台设备一个独立账号
- 某台设备丢失时单独停用
- 把某个账号限制到指定目录

创建 WebDAV 账号时可以：

- 自定义用户名
- 自定义密码，或让系统自动生成
- 指定根目录范围

密码只会在创建成功时显示一次，建议当场保存到密码管理器。

默认 WebDAV 地址是：

```text
https://你的域名/webdav/
```

如果管理员修改了 WebDAV 前缀，请改用新的地址。

## 设置

在设置页里，你可以调整：

- 主题模式
- 主题颜色
- 显示语言
- 文件浏览器默认视图

这些设置只影响你的前端使用体验，不会改动其他用户的数据。

# 服务器配置

```toml
[server]
host = "127.0.0.1"   # 监听地址
port = 3000           # 监听端口
workers = 0           # 工作线程数，0 = CPU 核心数
```

## 字段说明

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `host` | string | `"127.0.0.1"` | 绑定地址，Docker 中需设为 `0.0.0.0` |
| `port` | u16 | `3000` | HTTP 端口 |
| `workers` | usize | `0` | actix-web 工作线程数，0 表示自动检测 |

## Docker 环境

在容器中运行时，需要将 host 设为 `0.0.0.0` 才能从外部访问：

```bash
ASTER__SERVER__HOST=0.0.0.0
```
